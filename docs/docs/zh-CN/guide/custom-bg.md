## 自定义背景

在自定义下，可以选择第三方网络服务，自定义背景。

### API

适用于通过 API 接口获取图片地址，再从该地址下载图片的场景。

例如，某 API 返回结构如下：
```json
{
  "height": 1540,
  "ratio": "2367:1540",
  "url": "https://example.com/img380.webp",
  "width": 2367
}
```

表达式处填写 `$.url` 即可。

更多表达式语法，请参考 [JSONata 表达式](./jsonata.mdx)。


### 图片

适用于直接请求图片地址的场景（该地址可以使用 302 状态码进一步自动跳转）。

例如，访问某地址，响应如下：

![img](https://img11.360buyimg.com/ddimg/jfs/t1/397754/39/9503/20844/69a3d91bF41dcbc61/001526e1c8f0f1ac.jpg)

则可以在地址处直接填写该图片的地址。
